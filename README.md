# C Programming Basics 💻

This repository contains basic C programs for practice and learning.

## 📌 Programs Included
- Odd Even Check
- Prime Number
- Fibonacci Series
- Largest of Three Numbers
- Swap two Numbers
- Palindrome
- Multiplication table

## 🚀 How to Run
1. Compile using:
   gcc filename.c
2. Run:
   ./a.out
